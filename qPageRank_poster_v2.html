A Study of Ranking Dependence on the Deformation Parameter

Motivation & Background

Google's PageRank ranks web pages by computing the stationary distribution of a random walk on the web graph — equivalently the eigenvector of a stochastic link matrix for eigenvalue λ = 1.

Classical PageRank gives all outgoing links from a page equal weight, regardless of the structural importance of their destinations. This project asks a natural question:

What if transition probabilities are modified by a deformation parameter q, using q-numbers from quantum algebra? How does ranking change as q varies?

This introduces a one-parameter family of ranking models. The classical model is recovered exactly at q = 1, making q a natural dial between competing ranking philosophies.

Mathematical Construction (Section 4.3)

q-Number — recovers m as q → 1 via L'Hôpital:

[m]_q = (1 − q^m) / (1 − q)

Structural weight per node i (in-degree based):

φ_i = in-degree(i) + 1

q-Deformed web matrix — column-stochastic ∀ q > 0:

W_ij(q) = a_ij·[φ_i]_q/φ_i / Σ_ℓ(a_ℓj·[φ_ℓ]_q/φ_ℓ)

Google matrix with damping α = 0.85:

G(q) = α·W(q) + (1−α)·(1/N)·11ᵀ
Invertibility guaranteed: W(q) column-stochastic → λ_max = 1 always → α·1 = 0.85 ≠ 1 → det(I − αW(q)) ≠ 0 for all q > 0 and all α ∈ (0,1).
Three Natural Fixed Points

q interpolates continuously between three structurally distinct ranking regimes:

qRegimeWeight ∝
q → 0Anti-hierarchy1/φ — lower in-degree favored
q = 1Classical PageRankuniform — φ invisible
q → ∞Full hierarchyq^φ — higher in-degree dominates
Temperature analogy: q acts as inverse temperature. At q = 1, the φ hierarchy is completely invisible. Moving q away from 1 progressively reveals and amplifies it.

At q = 0: [m]_q = 1 for all φ, so weights reduce to 1/φ — the exact reversal of the φ ordering.

At q → ∞: highest-φ node in each column absorbs all mass — winner-takes-all in-degree ranking.

The (q−1) factor in every numerator is the algebraic mechanism guaranteeing classical recovery at q=1. It cancels with a matching (q−1) in each denominator, giving exactly 1/out-degree at q=1.
Key Conclusions
  • φ defines the landscape, q controls how much is expressed. Classical PageRank (q=1) is blind to the φ hierarchy.
  • Deformation is structurally selective — columns where all linked nodes share the same φ value are completely q-independent.
  • Polynomial degree of each entry is set by φ_max of the competing nodes in that column.
  • (q−1) cancellation in all numerators is the algebraic proof of classical recovery.
  • The (0,1) interval reverses the hierarchy — anti-hierarchy — and converges to its fixed point at rate O(q), faster than the q>1 polynomial rate.
  • 6 rank crossings detected: 4 in (0,1), 2 in (1,∞) — phase-transition-like events.
  • (I−αW(q)) is always invertible for all q>0, proven symbolically: no real positive roots of det=0.
  • q→∞ winner-takes-all: Node 5 rank jumps from 0.21 to 0.42 — pure in-degree hierarchy.
Methods & Implementation

Numerical (Python / NumPy):

  • Power iteration for PageRank (tol=1e-8)
  • Central finite differences for dr/dq (ε=1e-5)
  • Sign-change sweep for crossing detection
  • 300-point q grid over (0.01, 10)

Symbolic (Python / SymPy):

  • Exact rational W(q) via symbolic q-numbers
  • Classical recovery verified via sp.limit(q→1)
  • Exact r(q) as solution to linear system G(q)r=r
  • Exact dr/dq via sp.diff; exact crossings via sp.solve
Invertibility: W(q) column-stochastic → λ_max=1 always → α·λ_max = 0.85 ≠ 1 → (I−αW) invertible. Confirmed symbolically: det(I−αW(q)) has no real positive roots.
